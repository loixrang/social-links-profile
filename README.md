### What I learned

The important thing I learned during this project was "dvh" and "dvw" which means dynamic view height and dynamic view width respectively. This is a modern way of giving size to an element (the "body" in my instance) that makes it dynamic in both mobile and desktop devices, compared to the traditional "vh" and "vw".

Below are the code snippets:

```css
body {
  width: 100dvw;
  height: 100dvh;
}
```
